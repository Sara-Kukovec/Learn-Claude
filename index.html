Core Features You Should Know

These features turn Claude from a chatbot into a persistent workspace. Understanding them is the difference between casual use and real productivity.

Projects
Persistent workspaces with custom instructions and uploaded files. Claude reads everything before your first message. Create separate Projects for different contexts.
support.claude.com → How to Use Projects
Memory
Claude remembers preferences across conversations: your name, role, writing style, technical level. Builds automatically. View, edit, and delete in Settings.
support.claude.com → Memory FAQ
Artifacts
Live, interactive outputs: React apps, HTML pages, charts, documents, SVGs. Can call Claude's API, use persistent storage (20MB), and be shared via public links.
support.claude.com → What Are Artifacts
Web Search & Files
Claude searches the web for current information and creates files: Word docs, spreadsheets, PDFs, presentations. Upload documents for analysis or transformation.
support.claude.com → Web Search

Core Concepts

The principles that connect everything on this page. How Claude processes information, how you structure it, and how the discipline has evolved from writing prompts to designing complete AI environments.

The evolution: prompt engineering focuses on writing one instruction. Context engineering manages the full state across turns, tools, memory, and retrieval.

Finding the right altitude: too specific creates brittleness, too vague lacks guidance. The sweet spot gives strong heuristics without hardcoding logic.

Source: anthropic.com/engineering — Effective Context Engineering for AI Agents

Claude Skills

Skills are folders with instructions, scripts, and resources that Claude loads dynamically when relevant. They turn Claude from a general assistant into a specialist for specific tasks. Available across Claude.ai, Claude Code, Cowork, and the API. Open standard since December 2025.

Anatomy of a SKILL.md: metadata (name + description) is loaded at startup, body loaded on demand

Skills bundle additional files: Claude reads them only when the task requires it

Progressive disclosure: metadata → core instructions → nested resources, loaded level by level

How skills activate: system prompt → skill trigger → SKILL.md load → nested file load → task execution

Skills can include executable code: Claude runs scripts without loading them into context

Source: anthropic.com/engineering — Equipping Agents with Agent Skills

Built-in Skills
Claude ships with skills for Excel spreadsheets, PowerPoint presentations, Word documents, and fillable PDFs. These power the file creation feature in Claude.ai and Cowork.
anthropic.com → Introducing Agent Skills
Partner Skills
Pre-built skills from Atlassian (Jira, Trello), Canva, Figma, Notion, Cloudflare, Stripe, Zapier, and more. Install from the directory and use immediately.
anthropic.com → Skills Partner Directory
Custom Skills
Build your own: create a SKILL.md file with instructions, add scripts and resources. Use the skill-creator skill to generate them. Share via version control or the marketplace.
anthropic.com/engineering → Building Agent Skills
3-Level Loading
Metadata scanned at startup (few tokens). Core instructions loaded when relevant. Nested resources loaded on demand. Context-efficient by design.
